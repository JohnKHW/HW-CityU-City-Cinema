Event Handlers
On Ended Event
When at the end of video playing, wait for 2 seconds and switch to the next movie. The event handler will direct the event to next movie Which are scripted in ./index.html
On Change Event
When change the value of select option box The event handler will change the movie of movie list Which are scripted in ./now-showing.html
On Click Event
When click the seat of the seating plan The event handler will change the background of the colors Which are scripted in ./ticket.html
Script Structure Handlers
External: those scripts are repeated usage or remoted host, Inline scripts: event Handlers, Embedded: other scripts which are used and not included

CSS Group
There are a total of 10 style sheets (.css) and divide into 3 groups (Control style, Component style and Page style)

Control Style

  • webpage.css

Component Style

  • global.css
  • header.css
  • footer.css
  • layout.css
  • card.css

Page Style

  • index.css
  • now-showing.css
  • ticket.css
  • print.css

Control Style

webpage.css used for import all other .css file so we can write only one line in html file
<link href="css/webpage.css" rel="stylesheet" type="text/css" />

Component Style

Those are used for all pages common style.
e.g. layout, header, footer, card, block, content

Page Style

Those are used for all pages specific style to make sure the page is fit the design.

Header
Sticky Header
Used { position: sticky; } in header. 1. It makes the position of header fixed in its position so that they will be visible to the user forever. User does not need to scoll up or down to click the logo of header. 2. For header, { top: 0; } is used to position the header at the top of the page. Those codes are shown in "global.css" file at line 10 ~ 16 for header.
Disable Button
Used { cursor: default; } in disabled a. 1. It makes the cursor of a became in default cursor pointer Used { pointer-events: none; } in a. 1. It disable all pointer events of a such as hover, active etc. Those codes are shown in "header.css" file at line 67 ~ 80 for disabled a.
Footer
Bottom Footer
Used { min-height: calc(100vh - footer's height);} in main block. 1. It makes the position of footer always been the bottom of the page. Those codes are shown in "global.css" file at line 17 ~ 39 for header.
Center Footer Text
Used { text-align: center; } in footer. 1. It makes the position of footer's text always been the center of the page. Those codes are shown in "footer.css" file at line 1 ~ 5 for header.
